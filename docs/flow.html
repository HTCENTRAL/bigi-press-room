導入前の課題

  • 手入力の工数が大きく、担当者の負担になっていた
  • 情報をフォーマットに合わせて再入力する手間が毎回発生していた
導入前（アナログ）
導入後（システム）
✗ 手書き伝票
紙の伝票にブランド・商品情報・媒体・テーマを記入。手書きミス・紛失リスクあり
→
✓ フォームで登録
伝票番号は自動発行。入力ガイド付きでクラウドに保存
貸し出し
スタイリストへ商品を貸し出し
→
貸し出し（変わらず）
スタイリストへ商品を貸し出し
✗ 返却と掲載情報を手書き記入
どの媒体に・どのアイテムが採用されるか伝票メモで確認。スタイリストによって形式が異なる
→
✓ 返却と掲載を同時登録
伝票番号を入力するとアイテム一覧が表示され、1画面で完結
✗ Excelに転記
所定フォーマットのファイルを開き1件ずつ手入力。転記ミス・工数大
→
✓ 自動で転記
手入力・転記作業は一切不要。データが自動で蓄積。媒体種別、公開日順に整理され数秒で完成

導入でできるようになること

手入力をなくす

伝票起票・掲載情報転記作業をシステムが代替します。

情報の再入力が不要に

一度登録した情報が掲載リストに自動で反映されます。

情報を一元管理

貸出・返却・掲載のすべてが一箇所に。過去データもすぐ確認できます。
